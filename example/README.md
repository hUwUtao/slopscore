# Wheatbreeze Example

This folder contains a complete generated example from the project pipeline.

- `wheatbreeze.abc`: source notation with slot metadata and per-part reasoning
- `wheatbreeze.musicxml`: notation exchange file
- `wheatbreeze.mid`: MIDI playback file
- `wheatbreeze.mp3`: browser-friendly audio converted from the provided FLAC
- `wheatbreeze.opus`: smaller browser-friendly Opus audio
- `index.html`: minimal local preview page

The source FLAC was converted from `/home/stdpi/Downloads/medieval wheat.flac`.
