# Wheatbreeze Example

This folder groups generated examples by agent. The current example was
generated by `gpt5.5`.

- `gpt5.5/wheatbreeze.abc`: source notation with slot metadata and per-part reasoning
- `gpt5.5/wheatbreeze.musicxml`: notation exchange file
- `gpt5.5/wheatbreeze.mid`: MIDI playback file
- `gpt5.5/wheatbreeze.mp3`: browser-friendly audio converted from the provided FLAC
- `gpt5.5/wheatbreeze.opus`: smaller browser-friendly Opus audio
- `index.html`: minimal local preview page
